# django_react

## how to install
git clone https://github.com/geoffreymarche/django_react 

cd django_react 

bash install.sh

#### OR
git clone https://github.com/geoffreymarche/django_react 

cd django_react 

pip install -r requirements.txt 

npm install 

./node_modules/.bin/webpack --config webpack.config.js 

#### Launch the application

./manage.py runserver 

OR 

./manage.py runserver 0.0.0.0:8000
